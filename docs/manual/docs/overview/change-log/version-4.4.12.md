# Version 4.4.12

GeoNetwork 4.4.12 is a minor release.

**This release includes a security fix for a vulnerability affecting previous 4.4.x versions. All users are strongly encouraged to upgrade to 4.4.12 as soon as possible.**

## Migration notes

- [Map viewer / WFS filter / remove unused SLD API endpoints](https://github.com/geonetwork/core-geonetwork/pull/9343)
