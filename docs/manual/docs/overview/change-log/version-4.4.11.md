# Version 4.4.11

GeoNetwork 4.4.11 is a minor release.

**This release includes a security fix for a vulnerability affecting previous 4.4.x versions. All users are strongly encouraged to upgrade to 4.4.11 as soon as possible.**

## List of changes

- [Elasticsearch / Proxy / More robust aggregations processing](https://github.com/geonetwork/core-geonetwork/pull/9312)
- [Elasticsearch / Proxy / More robust filter processing](https://github.com/geonetwork/core-geonetwork/pull/9310)
- [Harden login redirect URL validation](https://github.com/geonetwork/core-geonetwork/pull/9309)
- [Library updates](https://github.com/geonetwork/core-geonetwork/pull/9308)
- [Metadata editor / close the service link dialogue box when linking it to the service metadata](https://github.com/geonetwork/core-geonetwork/pull/9295)

and more \... see [4.4.11-0 issues](https://github.com/geonetwork/core-geonetwork/issues?q=is%3Aissue+milestone%3A4.4.11+is%3Aclosed) and [pull requests](https://github.com/geonetwork/core-geonetwork/pulls?q=is%3Apr+milestone%3A4.4.11+is%3Aclosed) for full details.
