    Version 4.2.9

    GeoNetwork 4.2.9 release is a minor release.

    List of changes

    Major changes:

    • Create a metadata / Add dynamic and download privileges to the users in the same group
    • Decouple metadata user feedback from metadata rating feature
    • Extend http proxy to manage duplicated parameters
    • Fix MIME-types on attachments
    • Fix pdf link to the application website
    • Update org.json:json from version 20140107 to 20240205
    • Documentation / Manual improvements
    • Documentation / API SpringDoc fixes

    and more ... see 4.2.9 issues and pull requests for full details.
