Catalogue complet

Le catalogue des données et produits du Pôle ODATIS permettent de découvrir les jeux de données fournis par la communauté nationale et leurs liens avec les programmes internationaux.
Ce catalogue propose plusieurs outils de services d’accès aux données : un service de recherche avec des filtres de sélection, un service de description de la donnée (via deux onglets : "Aperçu" et "Complet"), un service de visualisation, et un service de téléchargement direct ou via le portail local des partenaires.

Plus d'information dans le Guide d'utilisation Sextant (édition IFREMER ) et dans une courte vidéo (53 secondes) appliquée à un jeu de données du Pôle ODATIS.
