GeoNetwork - Comenzando

Ejecutar la aplicación

GeoNetwork opensource es un Catálogo de Datos Espaciales. Para utilizarlo debe iniciar previamente el servidor web incluido.

En Windows, abrir a la carpeta GeoNetwork opensource, en el menú de inicio, que se crea durante la instalación y ejecutar "Start server". Entonces puede abrir GeoNetwork en su navegador web.

Por defecto, ha de acceder a http://localhost:8080/geonetwork

IMPORTANTE!! Después de abrir la página de inicio de GeoNetwork, validese como Administrador. Debe de usar el nombre de usuario admin y la clave de acceso admin. En el módulo de administración puede modificar su cuenta de usuario.

Puede iniciar/parar el servidor web Jetty de GeoNetwork usando los scripts ubicados en el directorio $INSTALL_PATH/bin.

En Windows:

  • startup.bat
  • shutdown.bat
En Linux o OSX:
  • startup.sh
  • shutdown.sh
