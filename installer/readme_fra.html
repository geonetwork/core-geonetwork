Lancement de l'application

  • aller dans le répertoire d'installation
  • puis dans le répertoire bin
  • utiliser startup.bat ou .sh pour lancer le catalogue
  • utiliser shutdown.bat ou .sh pour arrêter le catalogue
  • ouvrir la page http://localhost:8080/geonetwork dans un navigateur pour accèder au catalogue

Sous Windows, le menu démarrer présente également les options pour lancer/arrêter le catalogue.

L'identifiant admin et le mot de passe admin sont les identifiants par défaut.

Depuis la page d'administration il est possible de charger des modèles et exemples pour les standards de métadonnées installés.
